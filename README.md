# BDPortFlow Terminal Operating System UI

A full-stack web application for terminal operating system UI.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Start the backend server:
   ```bash
   npm run server
   ```

## Project Structure

- `src/components/`: React components
- `src/api/`: API client
- `src/server/`: Backend server code
